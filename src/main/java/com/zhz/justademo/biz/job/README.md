# 一个任务处理引擎

> 功能特点：
* 基于命令行
  * jat -r --register   注册任务
  * jat -r -- run       执行任务
  * jat -l --list       展示任务列表
  * jat 
* 支持http，rpc(dubbo)调用

基于数据库（PostgreSQL）的存储形式
